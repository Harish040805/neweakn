Emotion Aware Knowledge Navigator (EAKN)

The Emotion Aware Knowledge Navigator is an Emotion Detection and Productivity Tool designed to intelligently assist users in completing their tasks efficiently by analyzing the user's emotional state and dynamically arranging tasks based on their current mindset to help them complete work faster and with better focus.

Project Vision

EAKN aims to bridge the gap between emotion detection and task management. Instead of forcing productivity through rigid scheduling, the system adapts according to how the user feels, creating a more natural and effective workflow.

Core Features

  • Real-time Emotion Detection
  • Emotion-Based Task Prioritization
  • Smart Productivity Optimization
  • Dynamic Task Arrangement
  • Interactive Voice Assistance
  • Secure User Data Management
  • Repository connected with Database for Storage of files

How It Works

The system captures user emotion as input, processes it using AI-based emotion analysis, and categorizes tasks according to emotional compatibility. For example, analytical tasks may be prioritized during focused states, while creative tasks may be suggested during relaxed or positive moods. The integrated AI provides solution for the same tasks that are added.

Objective

The primary objective of EAKN is to enhance productivity while maintaining emotional balance. By aligning work with mindset, users can achieve better efficiency, reduced stress, and improved performance.
